## ?? Planning & Strategy

### Market Analysis
```

- Pricing strategy recommendations

```

## ?? Pro Tips

1. **Start broad, then narrow**: Begin with high-level requests, then drill down
2. **Reference files**: Always open relevant files before asking questions
3. **Be specific**: Include constraints, requirements, and preferences
4. **Iterate**: Refine outputs through multiple rounds
5. **Combine phases**: Use multi-step prompts for complex deliverables
6. **Validate**: Always review and test agent outputs
